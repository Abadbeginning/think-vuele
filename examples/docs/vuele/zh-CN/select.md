## Select

Select与el-select的使用完全一致；另外提供的几个扩展属性便于使用

### Attributes

| 参数          | 说明            | 类型            | 可选值                 | 默认值   |
|-------------  |---------------- |---------------- |---------------------- |-------- |
| providers   | 内容选项   | Array          | — | — |

#### providers

| name | 说明 |
|------|--------|
| id | 循环中的:key |
| text | 下拉框的文本 |
| value | 下拉框的值 |
