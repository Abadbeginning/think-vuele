# think-vuele

> vue+element进行优化封装的组件库，主要对标bootstrap的大集合UI框架limitless
> 框架结构按element源码结构进行改造

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8089
npm run demo

# build for production with minification
npm run dist
```
